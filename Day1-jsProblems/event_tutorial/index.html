누르면 alert가 나오는 버튼

오늘 진행할 실습에서는 이벤트 처리에 대해 배울거에요.

이벤트가 일어나는 이벤트 객체를 javascript에서 선택하고, 이벤트를 처리할 이벤트 핸들러를 구현하고, 이 둘을 addEventListener로 연결시켜주는 작업을 해볼 겁니다.


이 예제에서

<input type="button" value="버튼">이 클릭 이벤트가 발생할 이벤트 객체

alert_()가 클릭시 작동할 함수. 즉, 이벤트 핸들러입니다.


목표!

이벤트 핸들러인 alert_() 함수를 완성하고 이를 이벤트 객체와 연결해봅시다!

완성하면 저 버튼을 클릭할 때마다 "click"이라고 경고창이 뜰거에요.
